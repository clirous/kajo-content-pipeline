# Viral Content Patterns

Library of successful content patterns extracted from viral wellness posts. Grows over time as Stage 2 analyzes more content.

## Hooks

Patterns for grabbing attention in the first 3 seconds.

### Hook Types

1. **Question Hook** - Pose a thought-provoking question
   - Example: "Bạn có biết ánh sáng đỏ có thể..."

2. **Statistic Hook** - Lead with surprising data
   - Example: "80% người dùng báo cáo..."

3. **Problem Hook** - Address a common pain point
   - Example: "Mệt mỏi mỗi sáng? Có thể là..."

4. **Story Hook** - Open with personal narrative
   - Example: "Sau 6 tháng sử dụng..."

5. **Contrast Hook** - Before/after comparison
   - Example: "Từ mệt mỏi trầm trọng đến..."

### Vietnamese Hook Templates

```
- "Đừng bỏ qua nếu bạn..." (Don't skip if you...)
- "Sự thật về [topic] mà ít ai biết" (The truth about [topic] few know)
- "3 phút để hiểu [topic]" (3 minutes to understand [topic])
- "Tại sao [số liệu]% người [hành động]?" (Why [X]% people [action]?)
```

## Structure

Content structure patterns that drive engagement.

### High-Engagement Structure

```
1. Hook (3-5 giây đầu)
2. Problem identification (1-2 câu)
3. Solution/Insight (nội dung chính)
4. Evidence/Proof (nghiên cứu, số liệu)
5. CTA (kêu gọi hành động)
```

### Story Structure

```
1. Before situation (vấn đề ban đầu)
2. Discovery moment (khám phá giải pháp)
3. Transformation (quá trình thay đổi)
4. Results (kết quả đạt được)
5. Invitation (mời reader trải nghiệm)
```

### Educational Structure

```
1. Myth → Reality check
2. Science explanation (đơn giản hóa)
3. Practical application
4. Tips/Takeaways
5. CTA
```

## Tone

Voice and personality patterns that resonate.

### Primary Tone Elements

- **Empathetic**: Hiểu nỗi đau của reader
- **Authoritative**: Có căn cứ khoa học
- **Accessible**: Ngôn ngữ đơn giản, dễ hiểu
- **Optimistic**: Tập trung vào giải pháp, hy vọng

### Vietnamese Voice Guidelines

```
✅ Nên:
- Dùng "bạn" thay vì "quý khách"
- Câu ngắn, súc tích
- Kết hợp tiếng Anh khi cần (như "red light therapy")
- Sử dụng emoji vừa phải

❌ Tránh:
- Ngôn ngữ quá học thuật
- Câu dài, phức tạp
- Giọng quảng cáo quá đà
- Từ ngữ tiêu cực
```

### Persona: "be Ngo"

```
Giọng văn của be Ngo:
- Thân thiện, gần gũi
- Có kiến thức chuyên môn
- Chia sẻ từ trải nghiệm thực
- Không áp đặt, chỉ gợi mở
```

## CTAs (Call-to-Action)

Patterns for driving user action.

### CTA Types

1. **Learn More** - Khám phá thêm
   - "Tìm hiểu thêm về liệu pháp ánh sáng đỏ"
   - "Xem nghiên cứu đầy đủ tại [link]"

2. **Engage** - Tương tác
   - "Bạn đã thử chưa? Chia sẻ trải nghiệm bên dưới 👇"
   - "Tag người cần biết điều này"

3. **Save** - Lưu lại
   - "Lưu bài này để đọc sau"
   - "Screenshot để nhớ"

4. **Share** - Chia sẻ
   - "Chia sẻ để giúp nhiều người hơn"
   - "Gửi cho người đang cần"

### Vietnamese CTA Templates

```
- "Comment [từ khóa] để nhận [tài liệu]"
- "Follow để không bỏ lỡ tips sức khỏe"
- "Bạn nghĩ sao về [topic]? 👇"
- "Ai cần biết điều này? Tag họ 👇"
```
